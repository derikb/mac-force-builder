
MACS:
- class 1 (12pt), 2 (16pt), 3 (20pt)
- 1-Main gun (max class + 1)
- 2-6  (max 3 of same) (other guns <= class)
- max 2 brawl weapons must be same



AU:
- 1pt + #hardware + weapon class total per unit
- # in formation
- 2 max weapons
- max 2 dupes of same
- Infantry
  - can have Power 1 weapons
  - formations 1-5 units
- Vehicles
  - can have Power 2 weapons
  - formations 1-3 units


Weapons:
- Range: S, L, A
- Power: 1-4
  - MACs can have 1 weapon at (Class + 1) Power as Module 1, other weapons are limited to (Class) power.
  - AuxUnits
- Type: B, P, G, M
- Subtype: X, T, J, R
  - X can be compbined with other subtypes

Brawl:
- P2, B2, G2, M2
- plus optional subtype

Hardware:
- Types M/I/V/A
  - M: MAC only
  - I: Infantry only
  - V: Vehicle only
  - A: All types
- MACs cannot have more than three of the same module
