A web app for creating forces for <a href="https://chrismcdee.itch.io/mac-attack" target="_blank">MAC Attack</a>, a tabletop miniature game by Chris McDowall. It also has a play page for managing your force during a game.

Features:
- Create forces and save them locally to your browser (BACKUP OFTEN)
- Add MACs and Auxiliary Units to your Force.
- Tracks Force points total (even accounts for the -1 for a Framework module)
- Create Weapons.
- Add Hardware.
- Generate names for units and weapons.
- Set division and initiative suit/card for each unit.
- Checkboxes to track damage to modules and units.
- Mobile browser friendly.
- Import/export Forces.
